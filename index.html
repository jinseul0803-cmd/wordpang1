🔗 구글 시트 연동 (추천 · 학생 배포용)

선생님이 구글 시트에 단어·문장을 입력해 두면, 게임이 시트를 자동으로 불러옵니다.
시트만 수정하면 게임에 바로 반영돼요.
📋 시트 만드는 방법 (처음 1번만 설정)
  • 1단계. 구글 시트를 새로 만들고 아래처럼 입력합니다. (1행은 제목 행)
    • 구분영어뜻
    단어apple사과
    단어school학교
    문장I like apples.나는 사과를 좋아해요.
  • 2단계. 시트 메뉴에서 파일 → 공유 → 웹에 게시 → [게시] 버튼 → 링크 복사
  • (또는 오른쪽 위 공유 버튼 → '링크가 있는 모든 사용자 - 뷰어'로 바꾼 뒤 주소창 링크 복사)
  • 3단계. 복사한 링크를 위 칸에 붙여넣고 [시트 불러오기]를 누르면 끝!
  • 💡 학생 배포용 링크 복사를 누르면 시트가 연결된 게임 링크가 복사됩니다. 게임을 인터넷(학교 서버, 구글 사이트 등)에 올려 두었다면 학생들은 이 링크만 열면 선생님 단어장으로 바로 게임을 시작해요.

    • 🏆 우리 반 공유 순위판 (선택)

    앱스크립트 웹 앱 주소를 연결하면, 어느 기기에서 하든 점수가 하나의 순위판에 모여요.
    만드는 방법은 함께 드린 웹앱 만들기 가이드를 따라 하세요 (약 5분, 처음 1번만).

    ✏️ 직접 입력 (이 컴퓨터에만 저장)

    한 줄에 하나씩 영어, 뜻 순서로 쉼표(,) 또는 탭으로 구분해 입력하세요.
    엑셀에서 두 열을 복사해 그대로 붙여넣어도 됩니다.
